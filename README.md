POWER BI SALES DATA ANALYSIS PROJECT   
This project demonstrates a complete data analytics workflow including data cleaning and visualization using sales data.

## Phase 1 – Data Cleaning
- Raw Bike Sales dataset (.xlsx)
- Cleaned dataset after removing duplicates and handling missing values
- Data transformation and formatting

## Phase 2 – Data Visualization
- Interactive Power BI dashboard (.pbix)
- Sales performance analysis
- Regional and product-level insights
- Business conclusions derived from the cleaned dataset

## Tools Used
- Microsoft Excel
- Power BI
- Data Cleaning Techniques
- Data Visualization & Dashboard Design

## Objective
To convert raw sales data into meaningful business insights through structured preprocessing and interactive dashboards.
